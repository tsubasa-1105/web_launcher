# Web Launcher

パーソナル Web ダッシュボードツール。よく使う Web サイトを一個の画面で管理できます。

## 機能

- **リンクカードの追加・編集・削除**: お気に入りの Web サイトをカード形式で追加・管理
- **カスタマイズ**: カードの色や絵文字を自由に変更
- **ドラッグ＆ドロップ**: カードの並び替えが簡単
- **レスポンシブデザイン**: PC、タブレット、スマホに対応
- **デジタル時計**: ヘッダーにデジタル時計を表示

## 使い方

### 1. 起動

```bash
go run main.go
```

ブラウザで `http://localhost:8080` にアクセスします。

### 2. 設定モード

右上の「設定モード」ボタンをクリックすると、設定モードに切り替わります。

### 3. リンクを追加

1. 「新しいリンクを追加」ボタンをクリック
2. 名前、URL、色、絵文字を設定
3. 「変更を保存」ボタンをクリック

### 4. 既存のリンクを編集

1. 設定モードでカードの各項目を編集
2. 「変更を保存」ボタンをクリック

### 5. リンクを削除

1. 設定モードで削除ボタン（ゴミ箱アイコン）をクリック
2. 確認ダイアログで「削除」を選択
3. 「変更を保存」ボタンをクリック

### 6. 並び替え

設定モードで、カードのドラッグハンドル（⋮⋮）をドラッグして並び替えられます。

## データ保存

リンクデータは `data/links.json` に保存されます。

## 開発

### 依存関係

- Go 1.21 以上
- Tailwind CSS（CDN 経由）

### 構築

```bash
go build -o web-launcher main.go
./web-launcher
```
